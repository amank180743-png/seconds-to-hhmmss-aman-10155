## Project Description
This project converts a given number of seconds into hours, minutes, and seconds (HH:MM:SS format).

## Algorithm / Working
1. Take total seconds as input.
2. Calculate hours by dividing seconds by 3600.
3. Calculate remaining minutes using modulus.
4. Remaining seconds are printed directly.
